# MyPool - Система учёта занятий в бассейне

## О проекте

MyPool — это веб-приложение для автоматизации учёта занятий в бассейне. Система позволяет отслеживать информацию о бассейнах, тренерах, группах и абонементах, а также формировать аналитические отчёты.

### Функциональные возможности

- Управление бассейнами (CRUD)
- Управление тренерами (CRUD) с указанием дней работы
- Управление группами (CRUD) с категориями: начинающие, подростки, взрослые, спортсмены
- Управление абонементами (CRUD) с частотой посещений: 1, 2, 3, 5 раз в неделю
- Покупка абонемента клиентом (с уведомлением об оплате в клубе)
- Просмотр своих абонементов для клиента
- Аналитические отчёты (только для администратора):
  - Список тренеров по бассейнам
  - Итоговая прибыль каждого тренера
  - Тренеры, работающие с начинающими
  - Список посетителей по тренеру
  - Количество групп в бассейне по дням недели
  - Бассейн с максимальной выручкой

### Роли пользователей

| Роль | Возможности |
|------|-------------|
| Гость | Просмотр бассейнов, тренеров, групп, абонементов |
| Клиент | + Покупка абонемента, просмотр своих абонементов |
| Администратор | + Полный CRUD всех сущностей, аналитические отчёты |

## Установка и запуск

### Требования

- Node.js (версия 18 или выше)
- PostgreSQL (версия 14 или выше)
- npm или yarn

### 1. Клонирование репозитория

```bash
git clone https://github.com/mitjink/DoskovaMP_CourseProject  
cd DoskovaMP_CourseProject
```

### 2. Настройка базы данных

Запустите PostgreSQL и создайте базу данных:

```sql
CREATE DATABASE pool_db;
```

Выполните миграцию для создания таблиц:

```bash
psql -U postgres -d pool_db -f server/src/db/init.sql
```

Заполните базу тестовыми данными:

```bash
cd server
node src/db/seeds.js
```

### 3. Настройка бэкенда

Создайте файл `.env` в папке `server`:

```
PORT=8000
DB_HOST=localhost
DB_PORT=5432
DB_USER=postgres
DB_PASSWORD=ваш_пароль
DB_NAME=pool_db
JWT_ACCESS_SECRET=your_secret_key_here
JWT_REFRESH_SECRET=your_refresh_key_here
JWT_ACCESS_EXPIRES_IN=7d
JWT_REFRESH_EXPIRES_IN=30d
CLIENT_URL=http://localhost:3000
NODE_ENV=development
```

Установите зависимости и запустите сервер:

```bash
cd server
npm install
npm run dev
```

Сервер запустится на `http://localhost:8000`

### 4. Настройка фронтенда

Создайте файл `.env` в папке `client`:

```
REACT_APP_API_URL=http://localhost:8000/api/v1
```

Установите зависимости и запустите приложение:

```bash
cd client
npm install
npm start
```

Приложение откроется на `http://localhost:3000`
